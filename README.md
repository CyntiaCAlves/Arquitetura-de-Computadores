**Arquitetura de Computadores**

Este repositório foi criado para registrar meus estudos na disciplina de *Arquitetura de Computadores* durante o curso de _Análise e Desenvolvimento de Sistemas (ADS)_.

O objetivo é construir uma base sólida sobre como os computadores funcionam internamente, conectando teoria com a prática da programação.

**Objetivos**
- Compreender os conceitos fundamentais da arquitetura de computadores
- Entender como hardware e software se relacionam
- Desenvolver raciocínio técnico para programação
- Criar um material de apoio para revisões futuras

**Conteúdos abordados**
_Introdução_
- O que é Arquitetura de Computadores
- Hardware x Software
- Evolução dos computadores

_Representação de Dados_
- Bits e Bytes
- Sistema Binário
- Sistema Decimal e Hexadecimal
- Conversão de bases

_CPU (Unidade Central de Processamento)_
- Componentes da CPU
- Unidade de Controle
- ULA (Unidade Lógica e Aritmética)
- Registradores
- Ciclo de instrução

_Memória_
- Memória RAM e ROM
- Memória Cache
- Hierarquia de memória
- Armazenamento (HD e SSD)

_Sistemas e Comunicação_
- Dispositivos de entrada e saída
- Barramentos
- Sistemas operacionais

_Desempenho_
- Clock
- Processamento
- Paralelismo
- Pipeline

_Conexão com Programação_
- Relação entre memória e variáveis
- Execução de algoritmos na CPU
- Impacto da arquitetura no desempenho do software
